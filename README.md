# CMS
CMS with the internationalization in mind

## Dependencies
- [NodeJs](https://nodejs.org/en/download/)

## Installation
```bash
npm i
```

## Running the server

### For the production:
```bash
npm start
```

### For development
Use `--no-cache` flag to disable the caching.
```bash
npm start -- --no-cache
```

## Generating a static content
```bash
npm start
npm run static
```

## Test
```bash
npm start
npm test
```
